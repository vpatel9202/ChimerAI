# Milestones

This directory records ChimerAI roadmap milestones.

Milestone documents are planning and explanation artifacts. They should clearly
separate available behavior from planned behavior, and they should not replace
the task-oriented install and configuration guides.

## Current Documents

- [`0002-first-real-stack.md`](0002-first-real-stack.md): implemented Traefik,
  Authentik, OpenClaw, Todoist MCP, and bind-mounted state decisions.
- [`0003-public-alpha.md`](0003-public-alpha.md): release-readiness gates for a
  first public alpha.
- [`0004-mcp-and-agent-catalog.md`](0004-mcp-and-agent-catalog.md): planned MCP
  server and agent runtime expansion.
- [`0005-operations-maturity.md`](0005-operations-maturity.md): planned update,
  diagnostics, recovery, and operational maturity work.
