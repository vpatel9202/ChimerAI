# Milestone 7: Documentation and Discoverability

## Goals

- Make the README a concise first-60-second router.
- Add a compact architecture map for readers who need the system shape before
  deep docs.
- Add a sourced comparison guide for adjacent self-hosting projects.
- Add a demo and sample-output plan that does not fake artifacts.
- Make `docs/milestones/README.md` the single public roadmap source of truth.
- Update the agent context map so future agents can find the new docs.

## Release Gates

- README links to the canonical roadmap instead of duplicating milestone lists.
- Architecture map explains control plane, runtime plane, boundaries, and
  operator flow.
- Comparison guide cites official or project-maintained sources.
- Demo page distinguishes current evidence from planned artifacts.
- Agent context map includes the new public docs.

## Non-Goals

- No new product claims.
- No generated demo screenshots or fake command output.
- No broad competitor scorecard.
- No changes to private `.local/` content.
- No implementation of new runtime behavior.
