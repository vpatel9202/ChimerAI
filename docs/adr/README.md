# Architecture Decision Records

This directory records durable architecture decisions for ChimerAI.

ADRs should be used sparingly. Add one when a decision is hard to reverse,
likely to be debated, or important for future contributors to understand. Do
not add ADRs for routine implementation details, app-specific configuration,
small documentation changes, or choices that are still experimental.
